# Security Policy

Doria is experimental and pre-1.0.

Please do not report security issues through public issues if the issue could expose users to harm. Instead, use GitHub private vulnerability reporting if enabled, or contact the maintainer privately.

## Supported Versions

Only the latest commit on `main` is currently supported.

## Scope

Security reports may include compiler crashes, unsafe code generation, incorrect readonly/writable enforcement, dependency vulnerabilities, or repository supply-chain issues.
