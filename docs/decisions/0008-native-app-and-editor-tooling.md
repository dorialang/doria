# 0008 Native app and editor tooling

Status: Accepted

## Decision

Doria should support non-terminal native applications too. Major intended use cases include game engine editors and developer tools.

Doria should not merely clone Dear ImGui. The long-term UI and app direction should support polished layouts, dockable panels, inspectors, asset browsers, graph editors, timelines, themes, native platform integration, and embedding game or runtime viewports.

## Notes

Do not implement UI or runtime work now. This note records direction only.
